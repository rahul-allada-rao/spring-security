# Read Me First
The following was discovered as part of building this project:

* Spring Cloud Gateway requires Spring WebFlux, your choice of Spring Web has been replaced accordingly.
* The original package name 'arr.learn.spring-security' is invalid and this project uses 'arr.learn.springsecurity' instead.
